Retro Puzzle Game

A puzzle game I made as a sophomore. The player uses the arrow keys to move two characters at once. Avoid lava and moving enemies to get to the green goals. Have fun trying to find the fastest solution for each level!

Download!

I'm especially proud of this project because I worked with Govind Pimpale and Minh Phan to create a game that was both fun and genuinely challenging to complete. It has more polish than many of my other games, featuring retro music and a consistent style. On a less interesting note, this was the first time I made a game executable with a single .jar file
